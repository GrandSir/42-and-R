Assignment name  : ft_entab
Expected files   : ex01-21.c
Allowed functions: write
--------------------------------------------------------------------------------

Write a program entab that replaces strings of blanks by the
minimum number of tabs and blanks to achieve the same spacing. Use the
same tab stops as for detab. W

Input:
$> ./a.out "        selam"

Output:
Spaced Version:         selam

Tabbed Version: \t\tselam
